3.1 版更新历史

3.1.6 build 090919 ～ build 091012

  1. 词典页加入屏幕取词功能。
  2. 在笔记页中按 F10 将当前所选文字作为页面标题。
  3. 若干细节性改进与 bug 修复。
  4. 帮助文档加入索引与搜索功能。

3.1.5 build 090907

  1. 导入操作：支持从资源管理器中直接拖入文件或文件夹。
  2. 完善了整组的导入导出功能。现支持整组导出成文件夹, 以及整组从文件夹导入，两种操作均保留树结构。
  3. 按住 Ctrl 再点击“新建”，新建的节点将插入在当前节点之前。
  4. 改善了当从其它软件切换回 minipad2 时编辑区的焦点问题。
  5. 使用 Ctrl+Tab 可在编辑区与树／标签区之间切换焦点。
  6. 列表窗口：支持单击列头进行升／降排序，支持列头拖放以调整各列的顺序。
  7. 可设定用于发送邮件的邮件客户端程序。
  8. 计算页：支持百分号、中括号、大括号。（后两者将自动替换为小括号）
  9. 计算页：支持按 F9 进行整页重计算。
  10. 备忘页：弹出提醒框中新增“不再提醒”与“关闭后删除”选项。
  11. 文字模板：日期／时间格式自动遵从区域选项中设置, 并可自定义 %u(###)
  12. 文字模板：新增用于将任意编辑器中的选中文字添加到 minipad2 模板列表的快捷键设置。
  13. 链接页：支持从外部（桌面、资源管理器...）拖入链接。
  14. 调整了各对话框中的条目尺寸，修正了在 French、Spanish 等语言下对话框中条目文字显示不完整的问题。
  15. 其它小的完善性工作与 bug 修正。

3.1.4 build 090801

  1. 支持整组的导出与删除。
  2. 加入全局查找功能。
  3. 加入密码登录功能。

3.1.3 build 090724

  1. 完善了多语言支持，允许以外置 lng 文件的形式将软件翻译为除中英双语以外的其它语言。
  2. 完成了英文帮助文档。
  3. 加入了新选项：切换页面时焦点自动在编辑器。
  4. 对计算器、备忘录与文字模板作了少量的完善与增强。
  5. 一些小的完善与 bug 修正。

3.1.2 build 090707

  1. 修复了边缘隐藏时不停保存文件与无法重命名的 bug！
  2. 修复了调整窗口大小时闪烁的问题。
  3. 修复了当前行高亮下有时无法输入中文字符的 bug。
  4. 当窗口从边缘隐藏状态恢复时，编辑区自动获得输入焦点。
  5. 允许指定程序启动时载入的页面。
  6. 对中文帮助文档进行了 Html 重制作与编译。
  7. 一些小的完善与 bug 修正。

3.1.1 build 090630

  1. 编辑区块状文字选择后，支持使用 Tab 与 Shift_Tab 调整缩进。
  2. 编辑器中的链接可用 Ctrl_Enter 打开。
  3. 加入了 2.x 版的“记忆导出文件”选项。
  4. 加入多项树窗口的界面选项：显示水平滚动条、显示节点按钮、显示节点连线、根节点以线相连。
  5. 树窗口与标签栏均可选择是否显示图标。
  6. 在树窗口与编辑区的分隔条上增加了隐藏树窗口的按钮。
  7. 第一次启动时自动生成三项默认文字模板：插入日期、插入时间、插入日期时间
  8. 文字模板支持 Tab 与 Enter 的模拟按键，可用于自动填表。
  9. 剪贴板新增选项：弹出菜单宽度、弹出菜单无焦点、粘贴项目上浮到第一项
  10. 为文字模板、剪贴板与快速链接的弹出菜单增加了 ToolTip 提示。
  11. 剪贴板弹出菜单支持 Shift + 单击的合并粘贴。
  12. 加入“自动记录”功能，方便将 minipad2 作为资料收集面板。
  13. 备忘条目的提示音细分为 PC 发声与 wav 文件两种。
  14. 其它的一些细节性的完善与 bug 修复。

3.1.0 build 090614

  1. 加入了手动保存功能。
  2. 增强了剪贴板功能的稳定性。
  3. 修复了一些 bug。
  4. 完成了中文帮助文档。

3.1 beta5 build 090606

  1. 加入了“标签位于底部”选项。
  2. 加入了“操作习惯”系列选项。
  3. 加入了字数统计功能。
  4. 解决了编辑器文字与行号在某些字体下错位的 bug，以及在字体缩放时行号与文字会发生错位的 bug。
  5. 修改了自动备份的设计。原设计基于每隔若干小时进行备份，现设计为每隔若干次保存进行一次备份。
  6. 进一步完善了功能，修复了若干 bug。

3.1 beta4 build 090531

  1. 完善了拖曳支持。现支持标签页/组-->标签页/组（改变节点位置）、列表条目-->标签页（改变条目所属页面）、列表条目-->列表条目（改变条目排序）三种类型的拖曳。普通拖曳为移动，Ctrl+拖曳则为复制。
  2. 进一步完善了功能，修复了若干 bug。

3.1 beta3 build 090524

  1. 修复了较多的 bug，提高了软件稳定性。
  2. 修正了在非中文字体（如 Arial）下载入大文本时接近假死的问题。
  3. 加入了 3.0 版的“查看剪贴文字”功能。
  4. 为每一页面记忆光标所在位置，在载入页面时自动恢复。
  5. 计算页加入了 log(x,y) 内置函数（以 x 为底的 y 的对数）。

3.1 beta2 build 090520

  1. 基本修正了自 beta1 发布以来几天内发现的一些 bug。
  2. 增加了“外部滚动”选项。

3.1 beta1 build 090515

  1. 在 3.1 alpha 的基础上集成了 3.0 版的各种扩展功能。

3.1 alpha build 090421

本版相对于 3.0 版的更新：

  1. 源代码的大幅度结构调整与重写。
  2. 增加了对任意层目录的支持。增加“前一页”、“后一页”、“上一层”等导航操作。
  3. 增加了拖曳改变节点位置的功能。删除了不再有意义的“左移”、“右移”、“项目上移”、“项目下移”等按钮。
  4. 改变了数据存储结构。现为一对 idx + dat 的搭配。
  5. 计算器模块的重写与功能增强。现支持自定义函数与变量，支持注释。
  6. 集成了 2.3.2 版的“自动调整界面元素”功能。当缩小窗口时各界面元素会自动依次隐藏。
  7. 加入了高亮当前行选项。
  8. 查找与替换文字的输入增加下拉框，自动记忆历史记录。
  9. 增加了直接切换透明效果的菜单项。
  10. 参照 2.3.2 版设计，各扩展功能类似于插件，可在选项中选择是否启用。
