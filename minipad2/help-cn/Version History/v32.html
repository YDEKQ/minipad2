3.2 版更新历史

3.2.0 build 101003 .. 101128

  1. 再次调整了笔记小窗口的设计。省去了标题框，用@xxx代替标题
  2. 支持对各菜单条目快捷键的自定义
  3. 新选项：记忆页面滚动位置；树窗口位于右侧；分隔条宽度；关闭笔记热键、自动记录开关热键；剪贴板：过滤所有重复条目；
  4. 选项对话框中增加应用按钮
  5. 全局搜索增加“区分大小写”选项
  6. 文字工具增加删除空行、只空一行
  7. 丰富了字数统计的项目。
  8. Alt+Up, Alt+Down: 用于上、下移动所选项目（当列表窗口处于焦点时。仅支持单选），或在同一层次上向上、向下移动节点（当树窗口处于焦点时）。
  9. 升级了 3.1 版的各语言文件。
  10. 修正了一些 bug

3.2 beta5 build 100818

  1. 列表设定增加“应用到全部”按钮
  2. 支持摘要视图的字体与背景色设置
  3. 支持“新建笔记”小窗口的字体与背景色设置
  4. 备忘录加入年循环
  5. 剪贴板支持固定条目
  6. 快速链接与文字模板：加入在弹出菜单情况下用于快速存取某一项目的“缩写”项（模仿 win+r 风格）
  7. 文字模板新增 %c(##) 转义符，可用于打出任意的 ASCII 字符，包括不可见字符（适合于直接打出下划线高亮）
  8. 导出方式增加导出到剪贴板
  9. 在导入导出过程中，按住 Esc 可终止操作
  10. 其他小的改进、修复与性能优化

3.2 beta4 build 100718

  1. “插入链接”支持相对路径
  2. 计算器新增 int(x)（取整）, \(整除), !（取余）, round(x,y)（x 精确到 y 位小数）四个操作符／函数
  3. 列表的自定义支持拖放
  4. 恢复了“自动记录”功能
  5. ctrl_Left 向左隐藏／恢复树窗口，crl_Right 向右隐藏／恢复树窗口
  6. 全局搜索结果增加“节点路径”项目
  7. 新的 Ini 选项：定时提醒的默认时间
  8. 改进了托盘区“新建笔记”模块的设计，并可同时创建多个新笔记
  9. 其他小的改进和 bug修复

3.2 beta3 build 100526

  1. 更新了中英帮助文件。
  2. 一些小的改进与较多的 bug 修复。

3.2 beta2 build 100506

  1. 更新了英文语言文件。
  2. 新增了捕捉文字并保存为文件的功能。
  3. 完善了“插入链接”对话框；支持内部的节点链接。
  4. 较多的新选项、小的改进与 bug 修复。

3.2 beta1 build 100418

  1. 修改了局部文字高亮的设计。配色方案增加到4种，除下划线外还支持背景色。
  2. 修改了密码登录的设计。可设定托盘区锁定时间，可对原始的数据库文件进行加密。
  3. 重写了文件 I/O 的基础类，对于逐行读取大大提高了执行效率。
  4. 相比 alpha 的大量的 bug 修正与小的改进。

3.2 alpha build 100401

本版相对于 3.1.6 版的更新：

  1. 核心代码的完全重写与数据存储结构的重新设计。
  2. 菜单项目的重新整合。
  3. “属性”对话框，整合了与页相关的各种属性设置。添加了时间戳、外部图标、外部存储文件、备注等相关属性。
  4. 为列表类型的页增加了“摘要”视图。
  5. 任意一个笔记页均可选择是将数据存储于数据库文件中、还是存于独立的外部文本文件中。导入与文件拖入均支持外部存储方式。这使得 minipad2 可作为大量文本文件或电子书的树状结构的虚拟管理器使用，而无需将所有文本存储于单一的数据库中，大大分散了数据存取风险，提高了数据载入效率。
  6. 笔记导出支持 ANSI 等非 Unicode 编码方式。
  7. 系列“文字工具”：包括为文字添加下划线，以及添加列表符。
  8. 查找／替换功能的增强：支持在选区内查找，支持高亮查找结果。支持反向查找与替换。
  9. “插入链接”对话框，方便在笔记中插入文件链接。
  10. 取消“自动记录”功能，改为以全快捷键方式进行操作的“笔记摘录”功能。
  11. 增加“最近的笔记“导航项，包含最近创建、最近修改、最近访问三类。方便查找与切换到最近的笔记。
  12. 增加收藏夹功能。
  13. 增加全局搜索功能，可根据特定的标题、内容文字等进行搜索，列出所有匹配项目。
  14. 增加回收站功能。
  15. 简化了快速链接的设计。新版中的快速链接除了可以弹出菜单的形式显示以外，与普通链接页并无区别。
  16. 以列表形式进行剪贴板项目管理。剪贴板项目同样支持新建、编辑、删除等操作。
  17. 列表项目可显示勾选框（对于作为 to-do-list 使用的备忘页面较为有用）。部分列表项目（如联系人）支持独立的图标设置。
  18. 新增“统计信息”菜单项。不仅可对当前页面的文字进行统计，且可对整个数据库中各类页面的数量进行统计。
  19. “自定义工具栏”对话框中支持拖放操作。
  20. 完善了备份机制。可进行多重备份、可选择是定时备份、还是按一定的保存次数进行备份。
  21. 大量的新选项与其他小的改进。
