3.0 版更新历史

内核上的更新：

  1. 使用 Object Pascal / Win32 SDK 自行搭建 XCL 类库，在此基础之上构建 3.x 版。大大缩小了程序体积，提高了运行效率。
  2. 100% Unicode 内核，彻底根除乱码问题。
  3. 使用 RichEdit 代替 Edit 作为核心编辑控件，使用流式输入输出，极大地提高了编辑器的性能与对大文本的处理速度，且内置支持 OLE 拖曳、链接高亮、几乎无限次的撤销与恢复等新特性。
  4. 全新设计的数据存储结构，使用带索引的二进制块状文本读写取代原有的逐行读写与判断，极大地提高了数据存取效率。

功能上的更新日志：

final (2009-3-19)

  1. 行号选项
  2. 页边距选项
  3. 编辑器支持“追加”模式
  4. 增加了定时备份的选项
  5. 新增工具窗口风格（无任务栏按钮）、窗体透明度等选项
  6. 计算页新增角度弧度转换函数（deg、rad）
  7. 修正了几个小问题

alpha4 (2008-12-5)

  1. 加入集定时提醒、计划任务与日程管理功能于一身的备忘录功能。
  2. 集成 2.x 版的词典功能。
  3. 加入 2.x 版的自动隐藏功能。
  4. 文字内的超链接支持高亮特性。双击调用链接。
  5. 众多的其他小的改进。

alpha3 (2008-11-1)

  1. 集成2.3.2版的文字模板功能。
  2. 加入链接页（快启面板）、通讯录两项新功能。
  3. 支持标签页的跨组移动。
  4. 树窗口、列表窗口均可独立设置字体与背景色。
  5. 其他小的改进与bug修正。

alpha2 （2008-9-28）

  1. 集成 2.x 版的剪贴板监视。
  2. 三角函数支持角度制计算。支持 pi、e 两个常数（如 sin(e)+2+pi 是合法的）。支持在便笺页中的嵌入式计算。
  3. 众多小的改进，以及选项的细化。

alpha1（2008-9-14）

  1. 实现标签分组功能。支持任意多组标签，提供组之间的切换。
  2. 可在多页面标签风格与带树窗口的笔记风格之间进行切换。
