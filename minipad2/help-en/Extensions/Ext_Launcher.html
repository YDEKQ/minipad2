Launcher

The LinkPage is something like launcher or virtual-desktop, for manage of software links, file links & urls.
Use "File"-->"New ExtPage"-->"New LinkPage" to create a new LinkPage.

Basic Use

Use "Edit"-->"New Item" (or double-click on any empty place inside of linkpage) to open the "New Link" dialog to create a new Link.
(Another way to add new links is directly dragging the files / folders / shortcut icons from Windows Desktop / Explorer into the LinkPage of minipad2)
Right-click at existing item, choose "Edit Item" to open the "Edit Link" dialog to modify this item.
Double-click on existing items to launch the linked program/file/url...
To delete an existing item, right-click it and choose "Delete Item".
To adjust the order of items, select the item and drag it to the target position.
If there's more than one LinkPages, you can move/copy the links between different pages using dragdrop.

"Edit Link" dialog

Title: the title of the link

Category: including five kinds of link types:

  • Program/File: click the "..." button to associate a file. Execute this kind of link will open the associated program/file. If the program has parameters, please quote the program itself. e.g., "c:\notepad.exe" d:\123.txt
  • Folder: Use the "..." button to associate a folder. Execute this kind of link will open the associated folder.
  • Url: Input url in the "Link" (the single-line-edit). Execute this kind of link will open the default browser to launch the url.
  • Email: Input the email address in the "Link". Execute this kind of link will launch the default Email Client to create a new email, with the associated address as the receiver.
  • Batch Link: input multiple links in the "Link" (the multi-line-edit one). The links can be program/file, folder, url, or email address. Each link occupies one line. You can also use "Add" button to add one or more programs/files. Execute this kind of link will open the multiple programs/files/urls simultaneously.

Hotkey: The hotkey for quickly open this link.

Abbrev:the character / abbreviation used for quick launching this item in the popup menu (only for fast-links, to simulate win+r launching).

Fast Link

Fast link is a special link page. The links in this page can be launched via tray menu or popup menu.

Click "Tools"-->"FastLink Manage", you can launch the fast link page, in which you can add, delete, edit and double-click to run link items, just like in normal link pages. It also supports to drag in links from external source.

You can also launch the fast links from the tray menu.

Use the fastlink hotkey defined in "Options"-->"Extensions"-->"Launcher" to launch the fast link popup-menu. In the popup menu you can use mouse-pointer to select the target item, or you can also press 1..9 to directly launch the first nine items. If press Shift and click one item, all the items before the selected item will be simultaneously launched.

If "Enable subitem hotkey" is checked in "Options"-->"Extensions"-->"Launcher", then you can use the subitem-key to directly launch the first nine items. (e.g., if the hotkey for the fastlink is Ctrl+Alt+D, then the hotkeys for the first nine items will be Ctrl+Alt+1..9).

Options

In "Options"-->"Extensions"-->"Launcher" there're some options for the LinkPage, including Disable Icon-read, Minimize minipad2 after open link, FastLink Hotkey, etc. Please refer to the chapter "Options"-->"Extensions"-->"Launcher".
