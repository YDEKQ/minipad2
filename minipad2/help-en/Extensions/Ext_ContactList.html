Contact List

minipad2 has integrated the function of a very simple contact list.
Use "File"-->"New ExtPage"-->"New ContactPage" to create a new ContactPage.

Basic Use

Use "Edit"-->"New Item" (or double-click on any empty place inside of memopage) to open the "New Contact" dialog to create a new contact item.
Double-click an existing item to open the "Edit Contact" dialog to modify this item.
To delete an existing item, right-click it and choose "Delete Item".
To adjust the order of items, select the item and drag it to the target position.
If there's more than one ContactPages, you can move/copy the items between different pages using dragdrop.

Edit Contact dialog

In the "Edit Contact" dialog there're items including Name, Company, Address, Tel, Email, etc. It's quite easy to understand, so it's not necessary to give furthur descriptions here.
