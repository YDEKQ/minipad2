Template

minipad2 has integrated the function of text template. You can add many commonly used text or symbols into the template list, for paste and re-use under various circumstances.

Basic Use

Use "Tools"-->"Template Manage" to open the template-manage page. You can create, edit and manage templates in this page.
Use "Edit"-->"New Item" (or double-click on any empty place inside of the page) to open the "New Template" dialog to create a new template.
(Another way to add new template is: Select the target text in any editor, press the "Capture hotkey" which is set in "Options"-->“Template”, then the selected text will be captured (added) to the template list.)
Double-click an existing item to open the "Edit Template" dialog to modify this item.
To delete an existing item, right-click it and choose "Delete Item".
To adjust the order of items, select the item and drag it to the target position.

In the editor of minipad2, you can use "Edit"-->"Insert Template" to insert a template text into the current page.
In other windows editors, you can use the tray-menu of minipad2 to insert a template. You can also use the hotkey (which is defined in "Options"-->"Template") to launch the pop-up menu of template list. You can use mouse-click to select the items in the popup menu, or press 1..9 to directly launch the first nine items.
In the popup menu if you click an item while holding Shift, then all the items above this item (also including this item itself) will be simultaneously pasted.
For the paste of a certain template-item, you can use the hotkey defined for this item, or you can also use the subitem hotkeys for the first nine items, if "SubItem hotkey key" is checked in the "Options"-->"Template". (e.g., if the hotkey for the template popup is Ctrl+D, then the hotkeys for the first nine items will be Ctrl+1..9).

"Edit Template" dialog

Title: the title of the template. This title is only used for show in the template list, it does not represent the text being inserted.

Text: the template text which is used for insert into the editor.

Hotkey: the hotkey used for directly insert this template. (Optional)

Abbrev:the character / abbreviation used for quick launching this item in the popup menu.

Designating symbols

The "Text" part of the template supports the following designating symbols:

Designators:

  • %d: current date with the format of system's locale settings (short date).
  • %i: current time (no second) with the format of system's locale settings.
  • %u(###): DateTime with user-defined format. ### is the format string (e.g., yyyy-MM-dd HH:mm:ss). For more information of format strings, please refer to the settings and descriptions in your system's Control Panel - Local Settings.
  • %t: Tab-indent
  • %n: Carrige-return
  • %s(n): n continuous "-".
  • %s(###,n): n continuous ### (### can be anything).
  • %c(##): ASCII character ( including non-visual characters. ). ## is the ascii number. e.g., Tab: %c(9); carridge return: %c(13)%c(10)

Simulating keypress:

  • ~t: Tab
  • ~n: Enter
  • ~l: Left
  • ~r: Right
  • ~u: Up
  • ~d: Down
  • ~h: Home
  • ~e: End

you can add (n) after the above keypress-symbols to designate the repeat count of the keypress, e.g., ~l(5) will be pressing LEFT key for five times.

The software supports the mixture of normal text, indicating-symbols and simulating keypress. When the template is inserted, the indicating-symbols will be replaced with the text it represents, the simulating-keypress will be converted to the corresponding key-press (useful for input username-password or fill-in some tables).

Other special symbols:
  • ~c: indicating the caret position after insertion of the template.
  • ~s: indicating the selected text.

These two special symbols are useful for Html tag templates.

Examples

%d%i: when inserted it will be replaced with current datetime, like "2009-06-09 22:10"

%tFirst Item%n%s(*-,12)%n%tSecond Item: when inserted will be replaced with:
   First Item
*-*-*-*-*-*-*-*-*-*-*-*-
   Second Item

username~tpassword~n: when inserted will be:
username (Press Tab) password (Press Enter)

<p>~c</p>: after insertion will be:
<p></p>, and the caret will be between <p> and </p>.

<p>~s</p>: after insertion will add <p> in the left side of the selection and </p> in the right (will not replace the selected text).

<a ref="~s">~c</a>: Suppose you selected http://www.nebulasoft.cn in the editor and pasted this template, then the text after paste will be:
<a ref="http://www.nebulasoft.cn"></a>, and the caret will be in the left side of </a>.

Options

In "Options"-->"Template", you can set the hotkey for the template pop-up menu, and set whether to activate subitem hotkey or not. Please see the chapter "Options"-->"Extensions"-->"Template".
