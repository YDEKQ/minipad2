Extensions

The various extensions of minipad2 are like plug-ins which can be freely enabled/disabled (i.e., show or hide from the menubar and the toolbar) in this option page.

Enable CalcPage: set whether to activate the Calculator extension.

Enable MemoPage: Set whether to activate the Memo extension.

Enable DictPage: Set whether to activate the Dictionary extension.

Enable LinkPage: Set whether to activate the Launcher extension.

Enable ContactPage: Set whether to activate the Contact List extension.

Enable Template: Set whether to activate the Template extension.

Enable Clipboard: Set whether to activate the Multi-Clipboard extension.

Remark: The modifications in this page will take effect after the program is re-opened.

For the detailed descriptions of the various extensions, please refer to the corresponding chapters in "Extensions".
