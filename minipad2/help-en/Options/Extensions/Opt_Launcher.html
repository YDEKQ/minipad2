Launcher

Disable Icon-read: If set to "Yes", then for program/file links minipad2 will not read the separate icons, but use the same default icon for every link. This can improve the load-speed for the LinkPage (especially when there's some items link to the files on another computer through the intranet).

Auto minimize minipad2 after open link: set whether to minimize minipad2 after double-click and opening link. This is a useful feature for users to use minipad2 as program launcher.

FastLink Hotkey: Set the hotkey for FastLink pop-up menu.

Enable subitem hotkey: Set whether to register global hotkeys for the first nine FastLinks.
