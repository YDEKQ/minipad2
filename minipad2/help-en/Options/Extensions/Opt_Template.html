Template

Template Hotkey: Set the hotkey for the template-list popup menu. (You can use this hotkey to launch the pop-up menu of template list in any windows editor).

Enable subitem hotkey: Set whether to register global hotkeys for the first nine Templates.

Capture hotkey: Set the hotkey to "capture" (add) the selected text to the template list.

Template / Clipboard popup menu no focus: Set when use hotkey to launch the popup-menu, the input focus is still on the editor, not on the popup-menu. (the old 2.3/3.0 style)
Advantage: Can be used to edit filenames in Windows Explorer. If this item is set to "No", when the menu popups, the input focus will change to the menu, and the file-name-inplace-editor will disppear.
Disadvantage: Since there's no focus for the popup menu, it's not possible to use 1..9 to select the first nine items; Also it's not possible to use Esc to cancel the menu.
This settings is valid for the popup menus of both Template and Clipboard.
This settings is by default unchecked. i.e., the popup-menu has focus, can use 1..9 to select the first nine items, can cancel the menu with Esc.
