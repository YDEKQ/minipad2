Calculator

Decimals: Set the decimals (accuracy) for the calculation results. (for the values > 1, it's the digit-count after the decimal pointer; for the values < 1, it's the digit-count start from the first non-zero digit).
This value is adjustable between 0 ... 12, you can also set it as "float", which means there's no limit for the decimal of calculation result )

Triangular calc: Radian / Degree
If set to "Radian", then the triangular calculations ( e.g., sin(30) ) will based on radian, otherwise it will based on degree (i.e., 30 means 30°).
