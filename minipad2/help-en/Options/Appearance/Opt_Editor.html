Editor

Editor: Set the font ( including size, style & color ) and backcolor of the editor.

Margins: Set the width of the margin in both sides of the editor.

Show Line Number: Set whether to show line number on the left-margin of the editor or not. You can choose a color for the Line-number-bar.

Highlight current line: Set whether to highlight the current line (where the caret locates) or not. You can choose a color for the current-line-highlight.

Highlight scheme: Set the color & type ( underline or background highlight ) for find-result-highlight and four highlight schemes in "Text Tools".
