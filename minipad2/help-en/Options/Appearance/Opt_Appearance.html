Appearance

Languages: Choose the language of the program. (take effect after exit the "Options" dialog).

Tool-window style: set whether the main-window be tool-window-style (no "maximize" and "minimize" on the titlebar) or not.

Pagename on titlebar: if checked, then the text of titlebar will be something like "note1 - minipad2"; otherwise only "minipad2".

Show menubar: Show / hide the menubar.

Show toolbar: Show / hide the toolbar.

Splitter width: Set the width of the splitter (including the small button on it) between the tree-panel and the editor.

Transparency: Set the transparency of the main window. ( from 1 ... 10, where 10 is the highest transparency level ). To switch between opaque and transparent, use "Tools"-->"Transparent".

Auto adjust appearance: Let the program automatically adusts the visual elements ( menubar, toolbar, tabbar ... ) when the size of the window changed. Please see "Featured Functions"-->"Auto adjust appearance".
