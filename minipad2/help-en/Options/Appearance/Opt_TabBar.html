Tab Bar

Tab bar: Set the font of the tab bar (including font-size, font-sytle, font-color, etc.)

Tab on bottom: If set to yes, then the tabbar will be on the bottom. Otherwise it will be on the top side.

Multi-line tabs: If set to yes, then the tabbar will be multi-line style. Otherwise it will be single-line style.

Show tab icons: Set whether to show icons on the tabbar which indicates the page-type (Note, Calc, Memo...) of the tabpage.

Highlight current tab: Set whether to highlight the current tab.

Fixed tab width: Set whether the width of the tabs be fixed (adjustable from 1...10) or auto-adjusted (depending on the length of the title).
