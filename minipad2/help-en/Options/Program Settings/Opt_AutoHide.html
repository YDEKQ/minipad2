Auto hide

This option page contains the various settings for the Auto-hide feature.

Direction: the orientation for the side-hide. including four fixed directions (Left, Top, Right, Bottom) and one special "Free" direction. The default mode is "Free", with this mode the window will always hides to the side which is most close to the window. That means if the window hide in the right side previously and if you want it to hide to to top-side now, you needn't modify the settings, simply drag the window to a place close to the top-side of the desktop.

Hide delay: Set the delay time before hide, after the mouse-pointer leaves the window.

Show delay: Set the delay time for the window to show after the mouse-pointer reaches the side where the window hides.

Animation time: Set the speed of the window moving.

Side width: set the edge-width of the window when it hides in the side of the desktop.
