Notes

New note hotkey: Use this hotkey to launch the "New Note" dialog and write note. If there're text selected, then the selected text will be the title of the new note by default. In this dialog you can use "@xxx" to input the title of the new note in the first line.

Close note hotkey: Use this hotkey to save the new note and close the dialog ( same as click the close button of the dialog ).

New note in background: Use this hotkey to create a new note in background. If there're text selected, then the selected text will be the title of the new note by default.

Snap text: Use this hotkey to snap the selected text to the current note in minipad2.

New note in background and snap text: Use this hotkey to create a new note in background, and snap the selected text to this note.

Snap text to file: Press this hotkey will open the "Save to File" dialog, to save the selected text to the defined text file.

Switch auto record:Use this hotkey to enable / disable auto-record.
