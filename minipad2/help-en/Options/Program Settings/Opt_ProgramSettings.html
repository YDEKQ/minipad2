Program Settings

Start with windows: Automatically launch minipad2 when windows starts.

Show TrayIcon: select whether to show trayicon or not.

Minimize to tray: set whether minimize to the system tray (i.e., hide from taskbar when minimized) or not.

Minimize when close: Let the click on the "close" button in the titlebar equals to minimize the program.

Autoname new tabpage: Set whether to automatically name the new pages or not. If set to auto-name, then the new note pages will be named as note1, note2... the new calc pages will be calc1, calc2... the new memo pages will be memo1, memo2... the new dict pages will be dict1, dict2..., the new linkpages will be link1, Link2..., and the new ContactPages will be Contact1, Contact2... Later you can still use "Rename" to modify the page name.

Confirm delete: Set whether to show confirm dialog before page-delete or not.

External Scroll: Set whether to enable external scroll or not. (See "Featured Functions"-->"External Scroll")

Remember page scroll: Set whether to remember the scroll position for each page, and restore the position when switch back.

Always focus editor: If set to yes, then no matter under multi-page or treeview style, every time you switch to another notepage, the input focus will be automatically on the editor.

Recent notes: Set the maximum item count in the list of "Recent Create", "Recent Modify" and "Recent Visit".

Recycler limit: Set the maximum item count (not including the sub-nodes in the groups) of the recycler. When the total items exceeds this limit, the old items will be automatically cleared.

Minimize / Restore hotkey: set the hotkey for minimize / restore the program. ( use this hotkey to hide / call out minipad2 in any circumstance )

Mail Client: Set the mail client for creating new emails. If leave as blank, then when clicking "File”-->“Send Mail”, will load the system's default mail client to create the new mail.
