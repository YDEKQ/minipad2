Save / Backup

Auto save: Set whether to activate the timed save, and the interval for the timed save.

Auto backup: Set whether to enable the auto backup, and the backup interval ( time interval or save-count interval ).

Total backups: Set the maximum count of backups. During each backup the software will check the total backups, if it exceeds this limit, then the old backups will automatically be deleted.

Backup path: Set the path for backup.
