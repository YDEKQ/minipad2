Edit

Auto insert empty line: Let the editor automatically insert an empty line after each carriage-return. ( web-style )

Auto indent: Set the editor to be auto-indent style. ( the head of the new line automatically aligns with the head of the previous line after carriage-return )

Smooth scroll: When scroll the page content using the mouse wheel, whether to scroll smoothly or not.

One click open link: To open the links in the text, whether use single-click or not (otherwise use double-click or ctrl_click to open the links)

Confirm clear: Set whether to show confirm dialog before clear pagetext or not.

Show Line Number: Set whether to show line number on the left-margin of the editor or not. You can choose a color for the Line-number-bar.

Highlight current line: Set whether to highlight the current line (where the caret locates) or not. You can choose a color for the current-line-highlight.

Margins: Set the width of the margin in both sides of the editor.

Tab width: Set the indent-width when press Tab.

Undo limit: Set the maximum undo level (1..32767).

List 1, 2: Define the symbols for the "List 1" & "List 2" in "Edit"-->"Text Tools".
