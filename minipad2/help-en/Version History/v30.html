Version History
( 3.0 )

Updates of the kernel part:

  1. Rewrite the whole program using Object Pascal / pure Win32 SDK. Thus greatly reduced the exe size and enhanced the execution efficiency.
  2. 100% Unicode inside. Wrong encodings never appears anymore.
  3. Use RichEdit instead of Edit to be the kernel Edit control, use streamlized read/write mechanism, thus greatly increased the performance of the Editor especially for large text size. And now built-in support OLE dragdrop, link-highlight, unlimited undo/redo, etc.
  4. Fully new-designed data-storage structure, use indexed block-read-write instead of the original per-line read-write-decode, greatly increased the efficiency of data access.

Updates of the functionality:

3.0 final (2009-3-19)

  1. Support line number.
  2. Support page margins.
  3. Support "Protected" mode of the page content.
  4. Add features of backup.
  5. Add some new options like Tool-Window-Style, Transparency, etc.
  6. Add two new triangular conversion functions for the calc-page: deg、rad
  7. fixed some minor problems.

3.0 alpha4 (2008-12-5)

  1. Fully new-designed memo extension, which can be used as Reminder, Task Executor, or Scheduler.
  2. Integrate the dictionary extension of version 2.x.
  3. Add the auto-hide function of v2.x.
  4. The editor now support Link-highlight. Double-click to open the link.
  5. Some other minor improvements.

3.0 alpha3 (2008-11-1)

  1. Integrate the Template extension of v2.x.
  2. Add two new extensions: Link Page ( used as program launcher ), and Contact Page ( used as address list ).
  3. Support page-move across the groups.
  4. Support independant Font & BackColor settings for the tree window and list window.
  5. Other minor improvements and bug fixes.

3.0 alpha2 （2008-9-28）

  1. Integrate the multi-clipboard extension of v2.x.
  2. Some improvements for the Calc Page: The triangular functions (sin, cos...) now support to do calculations based on degree. Support two constants: pi & e（ e.g., sin(e)+2+pi ）. Support Embeded-calculation in normal Note Page ( use shift+enter by default ).
  3. Some other minor improvements and some new option items.

3.0 alpha1（2008-9-14）

  1. Support divide pages into different groups. Support unlimited group amounts, user can freely switch between different groups.
  2. Support switch between two styles: The traditional Multi-TabPage-Style and the new TreeView style.
