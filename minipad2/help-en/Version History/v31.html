Version History
( 3.1 )

3.1.6 build 090919 .. build 091012

  1. Add Tip-query function for the dictionary.
  2. In NotePage press F10 to use the selected text to rename the page.
  3. A few minor improvements and bug fixes.
  4. Add Index and Search support for the help file.

3.1.5 build 090907

  1. Support to import files by dragdrop files/folders from Windows Explorer.
  2. Improved group import/export. Now support export the whole group into a folder, or import folder into group. Both operations will keep the tree structure.
  3. Click “New...” while pressing Ctrl allows to create a new node which is inserted before the current node.
  4. Improved focusing of the editor while switching back from other windows.
  5. Use Ctrl+Tab to switch the focus between Editor and TreeView/TabBar.
  6. List Window: Support to sort the items by clicking on the column header. Support header dragdrop to adjust the orders of the columns.
  7. Support the definition of mail-client program for sending emails.
  8. Calculator: Support percent (%). e.g., 5% + 2 = 2.05
  9. Calculator: Press F9 will carry out re-calculate for the whole page.
  10. Memo: Add two new options: "No remind" and "Delete me" in the reminder's popup box.
  11. Template: The format of date / time now follows the local settings. It also supports user-defined format.
  12. Template: Add hotkey for capture the selected text into the template list.
  13. Launcher: Support to add new links by icon-dragdrop from desktop or Windows Explorer.
  14. Adjusted the item-size of the dialogs, fixed the text-cut problem under some languages (French, Spanish...)
  15. Other minor improvements and bug fixes.

3.1.4 build 090801

  1. Support the export & delete of whole group.
  2. Add global-search support.
  3. Add the option to allow login with password.

3.1.3 build 090724

  1. Improved the multi-language support, allows to translate the software into other languages outside of Chinese/English using external *.lng files.
  2. Finished the English help document.
  3. Add a new option: Always focus editor.
  4. A few improvements for the Calculator, Memo & Template.
  5. Some other minor improvements and bug fixes.

3.1.2 build 090706

  1. fixed a bug which caused the program to continuously saving file after auto-hide.
  2. fixed a bug which caused the flickering during window-resizing.
  3. fixed a bug which may cause sometimes unable to enter Chinese characters with "highlight current line" settings.
  4. When the window restored from Auto-hide, the input focus will now in the editor.
  5. Allow to define a page to load during program-launch.
  6. Other minor improvements and bug fixes.

3.1.1 build 090630

  1. Editor: Support to use Tab and Shift_Tab to adjust the indent of text block.
  2. Editor: Use Ctrl_Enter to open the url.
  3. Add "Remember export file" option of 2.3.
  4. Add some options for TreeView: Horz-scroll, Show node-buttons, Show node lines, Lines at root.
  5. Icon options for both TreeView and TabControl.
  6. Add a "Hide tree" button on the splitter between TreeView and Editor.
  7. Add three default templates: Insert date, Insert time, Insert date/time
  8. Template: Support KeyPress simulation of Tab & Enter. Can be used for auto-fill of forms.
  9. New options for the clipboard: Menu width, Popup menu no focus, New pasted item go to top.
  10. Add ToolTips for the popup menus of template, clipboard & fastlink.
  11. Clipboard Popup-menu: Support to use Shift + click for multi-paste.
  12. Add "Auto record" function, can be used for text collection.
  13. Reminder sound can be selected between beep and wav file.
  14. Other minor improvements and a few bug fixes.

3.1.0 build 090614

  1. Add "Save" menuitem for manual save immediately.
  2. Enhanced the stability of clipboard extension.
  3. Fixed some bug.
  4. Finished the Chinese help file. ( English help file will come in 3.1.1 )

3.1 beta5 build 090606

  1. Add the new option "Tab on bottom".
  2. Add a series of "Behavior" options.
  3. Add the Text-Count function.
  4. Fixed the bug of mismatching of the text and the line number under some fonts or after zoomin/zoomout.
  5. Changed the algorithm of auto-backup. While the old design is based on doing backup every predefined hours, the new design is now based on doing backup every a few saves (which is more reasonable).
  6. Furthur improved the functions and fixed a few bugs.

3.1 beta4 build 090531

  1. Improvement of the drag-drop support. Now support three different kinds of dragdrops: Page/Group --> Page/Group (change node position), ListItem --> Page (change the owner of the items), ListItem --> ListItem (change the order of the items). The normal dragdrop is move position, Ctrl + dragdrop is copy.
  2. Furthur improved the functions and fixed a few bugs.

3.1 beta3 build 090524

  1. Fixed a lot of bugs and improved the robustness of the program.
  2. Fixed the slow-speed problem of loading big-sized text under some non-system fonts.
  3. Add the "View ClipText" function of v3.0.
  4. Save and restore the scroll & cursor position for each page.
  5. Add a new internal function for the calculator: log(x,y). (logarithm of y based on x)

3.1 beta2 build 090520

  1. Fixed the bugs found after the release of beta1.
  2. Add a new option: "External scroll".

3.1 beta1 build 090515

  1. Integrated all the extensions of v3.0 based on the design of 3.1 alpha.

3.1 alpha build 090421

Updates compares to v3.0:

  1. Fully re-write of the source code.
  2. Support unlimited group depth (while 3.0 is flat-structure and only support one level of group). Add the navigation buttons: "Go Prior", "Go Next", "Level Up".
  3. Support change of node positions by dragdrop. Delete the no-longer useful menuitems: "Shift Left", "Shift Right", "Item Up", "Item Down".
  4. Re-designed the data-storage structure. With the new design the whole datafile is simplified to a pair of idx + dat. (in 3.0 each group is a single file)
  5. Re-write and enhancement for the calculator extension. Now support user-defined functions and variables, also support notation line.
  6. Integrated the "Auto adjust appearance" function of v2.3. When zoom-out the size of the window, the UI elements will disappear one by one...
  7. Add the optional feature of "highlight current line".
  8. Use DropDown ComboBox instead of a simple Edit control in the Find/Replace dialog, allows to re-select the historical find/replace strings.
  9. Add the menuitem to switch between transparent/opaque effects.
  10. All the extensions are now like plug-ins which can be freely enabled/disabled in the options. ( Just like the design of v2.3 )
