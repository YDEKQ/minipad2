Version History
( 1.0 ... 2.3.2 )

2.3.0 ... 2.3.2 ( 2007.12.2. ... 2008.4.22. )

  1. Re-write 90% codes.
  2. New clipboard-enhancement function. The clipboard menu can be called out in any windows editor program.
  3. New text-template function. The template menu can also be called out in any windows editor program.
  4. Delete the Desktop-pad mode, add two "Special Effect" modes ( "auto-scrollback", "auto-hide" ).
  5. The memo function now supports four modes: "Once", "Weekly", "Monthly" and "Daily". Add the pop-up effect of memo window.
  6. New menuitem / toolbutton for fast-switching between different pages.
  7. New "Send Email" function. The content of the pages now can be sent via email.
  8. New "Show Menubar" and "Auto adjust appearance" options.
  9. The toolbar is now user-definable.
  10. Support the right-click menu.
  11. Enhanced undo functions. Support the undo of "clear" and "replace".operations.
  12. New "Backup-center", allows to restore the page most recently deleted, and view the most recent two backup files.
  13. Numerous minor improvements and bug-fixings.

2.2.0 ... 2.2.1 ( 2007.9.23. ... 9.26.)

  1. New "Dictionary Page". The dictionary list is user definable and freely extensible.
  2. Support OLE dragdrop. Supports to drag the text from IE or other editors into the editor of minipad2. The dictionary page supports the dragdrop-style word-query.
  3. Supports two styles of "Desktop-pad" mode: "Auto-Scroll" and "Auto-Popup".
  4. Newly designed "Preference" window. Added many new options including "Auto empty line" and "export to previous file without prompt".
  5. The hotkeys for "Minimize/Restore" and "Desktop-mode" is now user-definable.
  6. The digits of calculation results is now user-definable.
  7. User can enable or disable the extended functions.
  8. Automatically remembers the window position when closed.
  9. Process-mutex design. Allows to run only one minipad2.exe process in the same directory.
  10. Supports extensible multi-language versions via auto-detection of *.lng files. Supports language-switching during program running.
  11. Other minor improvements and bug-fixings.

2.1.0 ~ 2.1.1 ( 2007.9.3. ~ 9.5.)

  1. New "Calculator Page".
  2. New "Memo Page"
  3. New "Desktop Pad" mode.
  4. Add windows hotkey for the minimize / restore and normal_mode / desktoppad_mode switching.
  5. Re-designed "Search / Replace" module, add "match whole word" option.
  6. Combined all the lesser-used options under "Options" menu into one "Preference" dialogbox. Add many new options like "Wordwrap", "Auto Indent", "Auto backup", "Minimize when close", "Fixed tab width", etc.
  7. A few bug fixing.

2.0.1 ~ 2.0.4 ( 2007.8.12. ~ 8.16. )

  1. Add standard notepad functions such as undo, select all, cut, copy, paste, etc.
  2. Allow exchange positions for the tab pages.
  3. New "Lock-Readonly-Unlock" switch function.
  4. Add search and replace function.
  5. Add import and export function for text files. The import function supports to import multi-files simultaneously.
  6. Auto-save when minimized.
  7. Add the option to switch between single-line-tab and multi-line-tab styles.
  8. Add transparancy settings.
  9. Add the Help file.
  10. A few bug fixing.

2.0.0 ( 2007.8.11. )

  1. The first version for minipad2 in it's true sense. Re-designed and re-write the whole program. Now it's designed to be a tiny, green, multi-page note program.

1.0 ( 2007.4. )

  1. The first version of minipad. A simple note program with dual pane (note list + editor) user interface.
