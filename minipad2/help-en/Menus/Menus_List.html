List

This menu is used for the List Pages (MemoPage, LinkPage, ContactPage, Template, FastLink).

New Item: Add a new item (Reminder, Link, Contact Person, ...) in the current page.
Insert Item: Insert a new item before the selected item.
Edit Item: Edit the selected item.
Delete Item: Delete the selected item.

Copy Content: Copy the content of the current list into clipboard. (Only copies the columns in the current view, do not copy the unactivated columns).
Define List: Set the columns, ViewStyle (Big Icon, Small Icon, List, Report...), Full-Row-Select-Style and GridLines for the current page. It's also possible to save the current settings as the default settings.
