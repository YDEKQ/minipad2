Define Accelerators

Minipad2 allows to re-define the accelerators for various menu items (including a few functions which only use accelerators without corresponding menu items).

How to re-define accelerators:

Create a text file named "accel.txt" under "minipad2\". Write into the file lines of “id=accelerator”. E.g.,

1010=Alt+W
1130=F7
1160=Shift+Ctrl+P
1720=

This will re-define the accelerator of "New Note” (id=1010) to Alt+W, “Switch Lock” (1130) to F7, “Property” (1160) to Shift+Ctrl+P, and remove the accelerator for “Stay On Top” (1720).
