Edit

Clear: Clear the content of the current page.
Undo: Undo the last edit.
Redo: Redo the cancelled edit.
Select All: Select all the contents in the current page.

Cut: Cut the selected text to clipboard.
Copy: Copy the selected text to clipboard.
Paste: Paste the content of clipboard to the cursor position.
Delete: Delete he selected text block, or selected list items.

New Item: Add a new item in the current page.
Insert Item: Insert a new item before the selected item.
Edit Item: Edit the selected item.

Word Wrap: Set whether the editor be word-wrap style or not.
Text tools: Drop-down menu, contains a series of text tools:
  Highlight 1 .. Highlight 4: Set underline or backcolor highlight for the selected text block. See "Options"-->"Appearance"-->"Editor" for the corresponding options.
  Remove Highlight: Remove highlight for the selected text block.
  Ordered List: Add numbers of order for the selected lines.
  List 1, 2: Add prefix of order for the selected lines ( by default is >, *. You can set your own prefix in "Options"-->"Program Settings"-->"Edit".)
  Remove List: Remove numbers or prefix of order for the selected lines.
  Delete all empty lines: Delete all empty lines for the selected text block (if no text block is selected, then delete all empty lines in the current page).
  Leave one empty line: For the selected text block (or the whole page if no text is selected), delete all empty lines, then insert one empty line between each paragraph.

Find / Replace: Open the "Find / Replace" window, input the text to find (and replace), set the find direction, whether to match whole word and case sensitive, whether only find in the selected text block, whether to roll-back, whether to highligh the match, and whether to exit this dialog after first match, then click "ok" to find the first match. The dialog also contains other buttons: "Find next" (if press Shift will be "Find Previous"), "Replace and find next" (if press Shift will be "Replace and fine previous"), "Replace all", for subsequent find and replace.
Subsequent find: Drop-down menu, contains the following items:
  Find Next: Find the next matching text. ( If set as "Down" and "Roll back", after reaches the end of the page will automatically scrolls back to the head of the page)
  Find Previous: Find the previous matching text.
  Replace & Find Next: Replace the current matching text and continues to find the next match.
  Replace & Find Previous: Replace the current matching text and continues to find the previous match.
  Replace All: Replace all the matching text in the current page.

Highlight Match: Highlight all the matching text in the current page.

Insert Link: Open "Insert Link" dialog, select link type (Program/File, Folder, Url, Node...), input the link text, then click "OK" to insert link in the current caret position. Use Double-Click (or Ctrl_Click) to open the link.
Insert TemplateText: Insert the selected template text to the cursor position. ( see "Extension"-->"Template" )
Insert ClipText: Insert the selected cliptext to the cursor position. ( see "Extension"-->"Multi-Clipboard" )
