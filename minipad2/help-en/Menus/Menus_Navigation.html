Navigation

Go Prior: Go to the prior page in the page history.
Go Next: Go to the next page in the page history.
Level up: Go to the parent group of the current page.

Recent Notes: Drop-down menu, contains the following menu items:
  Recent Create: Shows a list of the recent created notes.
  Recent Modify: Shows a list of the recent modified notes.
  Recent Visit: Shows a list of the recent visited notes.
  Manage Recent Notes: Add or switch to "Recent Notes" node in the tree window, to view or manage the recent notes.
Favorite: Show a list of notes in the favorite list. Contains the following two menu items:
  Add / Remove favorite: Add the current page to the favorite list, or remove from the list (if added).
  Manage Favorite: Add or switch to "Favorite" node in the tree window, to view or manage the favorite items.

Search: Open the "Search" dialog, select the page type (Note, Calculator, Memo...) and the field (Title, Content, Create Time...), input the search criteria, click "OK" to start searching. The search result will be shown as a list in the "Search" node in the tree window.
Recycle Bin: Click this item will add or switch to "Recycler" node in the tree window. All the items in the recycler will be shown as a list. You can move nodes out using dragdrop, clear the recycler, or (thoroughly) delete some items from the recycler.
