Tray Menu

The following items only appears in the tray menu:

New Note: Create a new note using mini-note window. In this window if the first line is started with @, then this line will be treated as the title of the new note.

Auto record: Open / Close the auto record function. Please refer to: “Featured Functions”-->“Snap Notes”。

Minimize / Restore: Minimize / Restore the main window.
