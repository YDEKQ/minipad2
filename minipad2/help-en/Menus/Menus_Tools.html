Tools

Show Tree: use this menuitem to switch between multi-page-style and tree-view-style.
Stay On Top: Let the minipad2 window to be the top-most window above all other windows programs. ( very useful for writing notes. )
Transparent: Switch the main-window between transparent status and normal status. (The transparency can be set in "Options"-->"Appearance").
Auto hide: Allows the window to automatically hides out of the desktop side, when the mouse pointer leaves the window. ( see "Auto Hide" section in the chapter "Featured Functions" )

Template manage: manage the template list. (see "Extensions"-->"Template")
FastLink manage: manage the fast links. (see "Extensions"-->"Launcher")

Watch Clipboard: Starts / Stops clipboard capture.
Clear Clipboard: Clears all the cached clipitems.
Clipboard manage: manage the clipitem list. (see "Extensions"-->"Multi-Clipboard")

Statistics: Open the "Statistics" dialog, in the "Current Page" tab you can see the count of characters, words, lines & paragraphs in the current page. In the "Database" tab you can see the count for different page types and items in the whole database.

Define Toolbar: Open the "Define Toolbar" dialog to define the visible buttons on the toolbar. In the dialog the right side is a list of current tool buttons, left side is the list of available buttons, it is possible to drag items between these two lists.
Options: Open the "Options" dialog to view/change the settings of the program. (see the chapter "Options")
