选项

程序设置：
	词库文件：选择词库文件。
	启用／停止热键：设置启动／停止单词切换（在常规模式下相当于显示／隐藏悬浮窗口）的全局热键。
	全屏模式热键：设定全屏模式的切换热键。
	鼠标跟随热键：设定鼠标跟随功能的切换热键。
	随系统自动启动：设置是否在系统启动时自动运行本软件。

界面设置：
	显示模式：在“常规”与“嵌入”两种显示模式间进行选择。
	单词与释义分行显示：选择释义是否另起一行显示（如否，则为“单词：释义”的形式。另如果词库中释义包含多行的话，则永远为分行显示）
	字体与背景色：选择常规模式下的字体与窗体背景色。
	窗体透明度：选择常规模式下窗体的透明度（0 .. 10，0 为完全不透明）。
	鼠标跟随间距：控制鼠标跟随状态下，单词窗口与鼠标位置之间的 x 距离与 y 距离（均为像素单位）。

词盒设置：
	词盒容量：设置词盒容量，即每一词盒中包含的单词数。
	自动切换词盒，每 X 个循环：设置是否在每一词盒进行了 X 次循环显示后，自动切换到下一词盒。
	第 X 个循环后自动转入测验模式：设置是否每一词盒在进行 X 次循环显示后自动进入测验模式。
	随机选取词盒：设置在更换下一词盒时是按词库中的顺序取下一词盒、还是随机任取一词盒。
	随机切换单词：设置切换单词时是按照单词在词库中的顺序依次取词、还是随机取词。
	单词切换间隔：设置切换单词的时间间隔。
	测验模式：设定测验模式的缓出方式。包含释义缓出、单词缓出、随机三种可选的缓出方式。
	测验模式释义显示滞后：设置在测验模式下，词条的部分显示与完整显示之间的时间差。

关于显示模式：
软件支持悬浮窗口、全屏与嵌入式三种显示模式。其中悬浮窗模式支持鼠标跟随风格。在悬浮窗模式下按 Esc 可隐藏主窗口（相当于在菜单中取消“启用”）。悬浮窗模式与全屏模式可通过菜单项或快捷操作自由切换，在悬浮窗模式下双击可进入全屏模式，在全屏模式下再次双击（或按 Esc）可退出全屏状态。嵌入式指的是无窗口、直接将词条显示在桌面当前窗口的标题栏上，这种显示模式需要在选项中进行设定。

关于词盒：
由心理学理论与实际学习经验可知，对于单词的记忆，如果仅在海量词库中随机取词、不作一定程度的重复记忆的话，记忆效果并不佳，短暂记住的单词也会很快忘记。
针对这一问题，本软件在设计时将海量词库拆分成一个个称为“词盒”（名称来源于装有一定数量卡片的小盒，每张卡片为一个单词）的逻辑单元，词盒容量（卡片数）可自行设定（默认为包含 50 单词）。在记忆时先反复循环显示某一词盒中的单词，可待记忆较熟后再手动切换至下一词盒（也可在选项中设成循环一定次数后自动切换），这样能大大提高单词记忆的效果。

关于测验模式：
测验模式是一种特殊的单词显示方式。常规的单词显示是同时显示词条与释义，而在测验模式下，将先只显示词条或释义，隔上一定的时间差后再显示全部内容。缓出方式与时间差可在选项中进行设置。

如何编辑词库：

云影单词的词库文件为纯文本文件，编码为 UTF16LE（Unicode），后缀名为 *.dwd。可使用记事本一类的纯文本编辑器进行编辑。

词库格式如下：
第一行：词条总数
以后各行：单词 (Tab) 释义

每条单词占一行。如释义内容本身为多行，用 \n 代替回车符。如释义中包含 Tab 缩进，用 \t 表示 Tab 符号。
以 // 起头的行将作为注释行，不加入单词列表中。（第一行词条总数后也可以 // 的形式加入注释）

特殊应用：

除了用于背单词外，云影单词尚可用于如下特殊应用：

1. 作为速读训练软件。
只需编辑与配置合适的词库，即可利用云影单词的全屏显示功能与灵活多样的单词切换设置，进行快速阅读训练。

2. 作为带自动切换功能的诗词、名句赏析软件
只需编辑与配置合适的诗词库，即可利用云影单词的迷你悬浮窗口、全屏模式与自动切换显示功能，作为诗词赏析软件使用。在工作与学习之余忙里偷闲，放松心灵，随时随地享受诗词的乐趣与美感。

3. 作为单词屏保
云影单词的全屏模式可作为屏保使用。在选项中可设置全屏模式的切换热键。这样当离开电脑时，只需按一下热键，即可将屏显切换为云影单词的全屏背单词状态。
